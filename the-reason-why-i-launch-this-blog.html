The reason why I launch this blog.

Date 二 20 五月 2014

I have tried to write a blog with the help of Flask, however I think it's a time-consuming work. And the GAE is so stingy that the dynamic site can't be support freely if the traffic of the site increase.

I don't want to spend much time on a personal blog, so I choose Pelican, a nice opensource static site generator written in Python.

I adopt Github Page to host the html files of this site.

I'll post some of my ideas, thoughts and academic related things on this site, aperiodically.
